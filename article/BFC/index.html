BFC

形成BFC的元素：
  • floats 不是 none
  • absolute positioned elements absolute／fixed
  • block container 但不是 block box的
    • 我觉得他们生成BFC的原因是因为 他们的内部都会形成一个透明的 block box。所以他们包含的有block-level box，就生成BF
  • inline-block ，其他的table-cell ，与table相关的先不说。
  • block boxes 的 overflow属性不是hidden的
  • block container 的内部子元素有block-level box的生成BFC
规则
  • 子元素的排列
  • 从 containing block的top开始进行排列
  • box 垂直分布
  • 依据 formatting 方式，默认是left-to-right，就是left out edge 接触到 left edge of containing block.
  • 如果是right-to-left 就是right edges touch
  • 即使存在float 元素，也是left-to-right，但是inline-box的宽度会变窄。
  • box的关系 和inline-level 的关系是？
  • 浮动元素和 block-level box的关系 ，block-level box不管float进行布局，但是内部inline-level box会有影响，会直接围绕他进行布局。
  • 子元素的距离
  • 由margin决定
  • ajacent block-level box的margin collapse
